# Piste デザイン変更指示書

> 参考サイト: https://piste-i.com/
> 作成日: 2026-02-14

---

## 1. 概要

現在の「Tropical Resort（トロピカルリゾート）」テーマから、Pisteブランドのデザインに全面変更する。
Pisteは「ココロとカラダが整う」をコンセプトとしたフィットネス・ウェルネス施設であり、ミニマルかつ信頼感のあるデザインを採用している。

---

## 2. カラースキーム変更

### 2.1 変更対象: 全CSSファイル共通

| 用途 | 現在の値 | 変更後の値 | 備考 |
|------|---------|-----------|------|
| **メインカラー（Primary）** | `#00A86B`（エメラルドグリーン） | `#8dc21f`（黄緑/ライムグリーン） | Pisteのブランドカラー |
| **メインカラー（Strong）** | `#1A4C43`（深い緑） | `#6a9a18`（濃いライムグリーン） | ホバーや強調時に使用 |
| **メインカラー（Soft）** | `#E6E6E6`（ライトグレー） | `#e8f5d4`（淡いライムグリーン） | 薄い背景・ボーダーに使用 |
| **背景色** | `#FDF6E3`（サンド/クリーム） | `#ffffff`（白） | Pisteは白基調 |
| **背景色（セカンダリ）** | - | `#f5f5f5`（ライトグレー） | セクション区分に使用 |
| **カード背景** | `#ffffff` | `#ffffff` | 変更なし |
| **テキスト（Strong）** | `#143D36`（深い森グリーン） | `#333333`（ダークグレー） | Pisteの基本テキスト色 |
| **テキスト（Muted）** | `#4A665A`（ミュートグリーン） | `#666666`（ミディアムグレー） | 補助テキスト色 |
| **オーシャンブルー** | `#0077BE` | 削除 | Pisteでは未使用 |
| **パステルブルー** | `#81D4FA` | 削除 | Pisteでは未使用 |
| **バッジ背景** | `#F2F2F2` | `#f5f5f5` | 微調整 |
| **シャドウ** | `rgba(0, 119, 190, 0.12)` | `rgba(0, 0, 0, 0.08)` | ニュートラルな影に変更 |

### 2.2 CSS変数の変更マッピング

```css
/* Before (Tropical Resort) */
:root {
  --bg: #FDF6E3;
  --card: #ffffff;
  --brand-pink: #00A86B;
  --brand-pink-strong: #1A4C43;
  --brand-pink-soft: #E6E6E6;
  --brand-teal: #0077BE;
  --brand-light-blue: #81D4FA;
  --text-strong: #143D36;
  --text-muted: #4A665A;
  --badge: #F2F2F2;
  --shadow: 0 30px 60px rgba(0, 119, 190, 0.12);
}

/* After (Piste) */
:root {
  --bg: #ffffff;
  --bg-alt: #f5f5f5;
  --card: #ffffff;
  --brand-primary: #8dc21f;
  --brand-primary-strong: #6a9a18;
  --brand-primary-soft: #e8f5d4;
  --text-strong: #333333;
  --text-muted: #666666;
  --text-black: #000000;
  --badge: #f5f5f5;
  --shadow: 0 20px 40px rgba(0, 0, 0, 0.08);
}
```

### 2.3 対象ファイル

- `src/style.css`（ルータ画面）
- `src/form2/style.css`（アンケート画面）
- `src/generator/style.css`（口コミ生成画面）
- `src/admin/style.css`（管理画面）

## 3. タイポグラフィ変更

### 3.1 フォントファミリー

| 用途 | 現在 | 変更後 |
|------|-----|-------|
| **ベースフォント** | `Zen Kaku Gothic New`, `Noto Sans JP` | `Arial`, `Hiragino Kaku Gothic ProN`, `Yu Gothic`, `Meiryo`, sans-serif |
| **アクセントフォント** | `Outfit` | 削除（ベースフォントに統一） |

### 3.2 Google Fonts

- `Outfit` と `Zen Kaku Gothic New` の読み込みを**削除**
- 代わりにシステムフォントのみ使用（読み込み速度向上）

### 3.3 フォントサイズ

| 要素 | 現在 | 変更後 |
|------|-----|-------|
| **大見出し（h1）** | `clamp(1.5rem, 5vw, 2.2rem)` | `clamp(1.8rem, 5vw, 2.5rem)` |
| **ボタンラベル** | `clamp(1.3rem, 4vw, 1.8rem)` | `clamp(1.1rem, 3.5vw, 1.4rem)` |
| **本文** | `0.85rem～0.95rem` | `0.9rem～1rem` |

### 3.4 フォントウェイト

- 見出し: `700`（太字）→ `500`（ミディアム、Pisteに合わせてやや軽め）
- 本文: 変更なし（`400`）

## 4. レイアウト変更

### 4.1 全体構造

| 項目 | 現在 | 変更後 |
|------|-----|-------|
| **ページ背景** | クリーム色 + 放射グラデーション | 白 `#ffffff`（フラット） |
| **カード角丸** | `clamp(32px, 5vw, 40px)` | `0px`（角丸なし、シャープ） |
| **カードボーダー** | `1px solid #E6E6E6` | `none`（ボーダーなし） |
| **カードシャドウ** | 大きなソフトシャドウ | 最小限 or なし |
| **装飾画像** | `coral.png`, `mangrove.png`（透過表示） | **削除**（Pisteは装飾なし） |

### 4.2 背景装飾の削除

以下の疑似要素を削除:
- `.router::before`（coral.png）
- `.router::after`（mangrove.png）
- `body::before`（放射グラデーション）

### 4.3 カード → フルワイドレイアウト

Pisteのデザインはカード型ではなく、フルワイドで構成されている。

```
現在: body → 中央配置カード（max-width: 520px）
変更: body → フルワイド（max-width: 100%）、コンテンツは中央揃え
```

## 11. デザイン変更の優先順位

| 優先度 | 項目 | 理由 |
|--------|------|------|
| **P0（必須）** | カラースキーム変更 | ブランドアイデンティティの基本 |
| **P0（必須）** | ブランドテキスト/タイトル変更 | `ISHIGAKIJIMA ACTIVITYCLUB` → `Piste` |
| **P0（必須）** | form2の日本語化 | 英語→日本語への全面変更（HTML・JS両方） |
| **P1（高）** | ボタンスタイル変更 | ユーザーが最も触れるUI要素 |
| **P1（高）** | 装飾画像の削除 | トロピカルテーマとの決別 |
| **P1（高）** | フォント変更 | 全体の印象を大きく左右 |
| **P2（中）** | レイアウト変更（角丸→シャープ） | デザインの一貫性 |
| **P2（中）** | ロゴ画像の差し替え | ロゴ素材の入手が前提 |
| **P3（低）** | 管理画面のカラー変更 | ユーザーに見えない部分 |
| **P3（低）** | アニメーション調整 | 微細な改善 |

## 12. 備考・確認事項

1. **ロゴ画像**: Pisteのロゴ画像（SVG推奨）の提供が必要
2. **ファビコン**: Pisteのファビコン（16x16, 32x32, 180x180）の提供が必要
3. **コピー文**: トップページの見出し・説明文の最終テキストについてPiste側での確認が必要
4. **レスポンシブ**: Pisteのサイトはモバイル時に黒背景ヘッダーを採用しているが、本アプリでは白背景で統一する方針でよいか確認が必要
